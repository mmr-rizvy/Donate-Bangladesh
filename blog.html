What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a document (usually HTML or XML) as a tree of objects, where each node in the tree corresponds to a part of the document, such as elements, attributes, and text.

How do you select an element from the DOM?

You can select elements from the DOM using various methods provided by the Document Object Model (DOM) API in JavaScript. These methods provide a powerful way to select and manipulate elements in the DOM, enabling you to create dynamic and interactive web applications. Use querySelector and querySelectorAll for more complex selections using CSS selectors, while getElementById, getElementsByClassName, and getElementsByTagName can be handy for simpler cases.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in the DOM where a single event listener is attached to a parent element instead of adding individual event listeners to each child element. This approach leverages the concept of event bubbling, where events propagate up the DOM tree, allowing the parent element to catch events from its children.

How do you manipulate an element's attributes and styles using the DOM?

Attributes: Use setAttribute, getAttribute, removeAttribute, or direct property access to manipulate attributes.
Styles: Use the style property to manipulate styles directly or use classList for adding, removing, or toggling CSS classes.
These methods allow you to create dynamic and interactive web applications by altering the appearance and behavior of elements on the fly.
